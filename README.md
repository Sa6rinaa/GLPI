# GLPI - Gestionnaire Libre de Parc Informatique

## Résumé du Projet

GLPI (Gestionnaire Libre de Parc Informatique) est un logiciel libre open-source dédié à la gestion des services informatiques et d'assistance. Il permet de gérer efficacement un parc informatique en offrant des fonctionnalités pour l'inventaire, la gestion des tickets d'assistance, la gestion des contrats, et bien plus.

### Fonctionnalités Principales
- Gestion des actifs informatiques (ordinateurs, serveurs, périphériques, etc.)
- Système de tickets pour la gestion des demandes d'assistance
- Gestion des contrats et fournisseurs
- Rapports et statistiques
- Interface web intuitive

### Installation
Le projet inclut un guide détaillé pour l'installation de GLPI sur un environnement LAMP (Linux, Apache, MariaDB, PHP). Cela comprend :
- Configuration de machines virtuelles (Debian et Windows)
- Installation et configuration du serveur LAMP
- Déploiement de GLPI et configuration de la base de données

Pour plus de détails, consultez le fichier `GLPI.md` qui contient un compte rendu complet des étapes d'installation et de configuration.
